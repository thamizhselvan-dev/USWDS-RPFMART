## Introduction

This USWDS-RPFMART project is a responsive Home Page built using USWDS-based HTML templates. It showcases my projects, skills, and professional experience with a focus on accessibility, usability, and modern design standards. The application is designed to be user-friendly, visually appealing, and fully responsive across devices.

## Features

- Built on U.S. Web Design System (USWDS)
- Easy to customize and extend Components.
- Accessibility-first and WCAG-aligned.
- Responsive Design: Works on all devices.
- Contact Form: Allows visitors to get in touch with me directly through the site.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/thamizhselvan-dev/USWDS-RPFMART.git
   cd USWDS-RPFMART

   ```

2. Install the dependencies:

   ```bash
   npm install

   ```

3. Start the development server:

   ```bash
   npm start

   ```

4. Open your browser and navigate to http://localhost:8080

## Usage

To use this portfolio:

1. Customize the content in the `src` directory to reflect your personal information and projects.
2. Update the `src/**` directory to adjust the structure and layout as needed.
3. Deploy the application to your preferred hosting service (e.g., GitHub Pages, Netlify, Vercel).

## Technologies Used

- **UDWDS**: Design system and JavaScript components for building accessible, government-standard user interfaces..
- **Swiper**: Library used to implement responsive sliders and carousels in the application.
- **Styled Components**: Library for styling USWDS components with reusable, scoped styles.
- **Bootstrap CSS**: Integrated for layout utilities and responsive design support.
- **GitHub Pages**: Hosting services for deployment.
